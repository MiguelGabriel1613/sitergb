RGB vs CMYK - Como Funcionam?

O que é RGB?

O modelo RGB é um sistema aditivo de cores utilizado em telas. Ele combina as cores Vermelho (Red), Verde (Green) e Azul (Blue) para formar outras cores.

O que é CMYK?

O modelo CMYK é um sistema subtrativo de cores usado na impressão. Ele usa as cores Ciano (Cyan), Magenta, Amarelo (Yellow) e Preto (Black) para criar diferentes tonalidades.

Diferenças entre RGB e CMYK

O modelo RGB é usado para telas e trabalha com luz, enquanto o CMYK é utilizado para impressão e funciona com pigmentos.
