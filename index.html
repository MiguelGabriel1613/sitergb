RGB vs CMYK - Como Funcionam?

🔴🟢🔵 O que é RGB?

O modelo RGB é um sistema de cores usado em telas de celulares, computadores e TVs. Ele combina três cores básicas: Vermelho (Red), Verde (Green) e Azul (Blue). Essas cores somadas criam milhões de outras tonalidades!

🖨️ O que é CMYK?

O modelo CMYK é usado em impressoras para criar cores em papel. Ele mistura Ciano (Cyan), Magenta, Amarelo (Yellow) e Preto (Black) para formar imagens coloridas.

🎨 Diferenças entre RGB e CMYK

A maior diferença entre os dois sistemas é que o RGB trabalha com luz (ideal para telas), enquanto o CMYK trabalha com tinta (ideal para impressão). No RGB, quanto mais cor adicionamos, mais claro fica. No CMYK, quanto mais cor adicionamos, mais escuro fica.

💡 Curiosidade

Você sabia que nossas telas nunca mostram a cor preta de verdade? O preto que vemos é, na verdade, a ausência de luz! No RGB, quando todas as cores estão desligadas, vemos preto. Já no CMYK, o preto real é criado com tinta especial para evitar desperdício de cores.
